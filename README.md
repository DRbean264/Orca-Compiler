# Orca Compiler

This is a Tiger compiler written in SML/NJ for ECE 553 Spring 2023.

## Getting started

```
sml sources.cm
```

At the sml prompt, you can play with the lexer by calling

```
Parse.parse "testcases/test1.tig"
```

## Running Tests

```
sml sources.cm test_lexer.sml
```
